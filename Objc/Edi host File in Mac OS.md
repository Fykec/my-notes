Edi host File in Mac OS {.heading-level2}
-----------------------

1.终端更改 这种好像是网路上说的最多的方法.
在应用程序里面打开终端(terminal),输入 sudo vi /etc/hosts\
 然后提示输入系统密码 hosts文件就自动打开了 接着输入 i 进入编辑模式
将添加的网站,ip拷贝进去 编辑完成之后,按esc,输入 :\
 wq 这样就更改完成了.\
